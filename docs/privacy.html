Privacy Policy

Privacy Policy

Effective Date: November 22, 2025

At AxelBase UUID v4 Generator, your privacy is not just respected — it is guaranteed by design. This tool was built from the ground up to generate cryptographically secure UUIDs entirely in your browser, with zero data collection.

We Collect Nothing

We do not collect, store, transmit, or process:

  • Personal information (name, email, IP address)
  • Usage data or analytics
  • Generated UUIDs
  • Browser or device fingerprints
  • Cookies, localStorage (except optional theme preference)
  • Any form of telemetry or tracking pixels

100% Client-Side Execution

Every UUID is generated using your browser’s native crypto.randomUUID() or crypto.getRandomValues(). All formatting, copying, and exporting happens locally. No data ever leaves your device.

No Third-Party Tracking

The only external request is to the official Bootstrap CDN for styling — and even that can be self-hosted. We do not use Google Analytics, Meta pixels, Cloudflare insights, or any tracking service.

Offline-First & Air-Gapped Ready

After the first load, this tool works completely offline. You can disable internet access, block all scripts, or run it from a USB drive — it will still generate perfect RFC 4122 v4 UUIDs.

Data Security = Your Responsibility

Since we never see your data, security is fully in your hands. We recommend using a trusted browser and avoiding public networks when generating sensitive identifiers.

Children’s Privacy

This tool is safe and appropriate for all ages. No personal data is ever requested or stored.

Changes to This Policy

We may update this policy only to reflect technical improvements. Any changes will be clearly dated. Continued use constitutes acceptance.

Contact

This project is open source and community-driven. Questions? Open an issue on our GitHub repository.
