Privacy-First Design: No Tracking, No Analytics

Published: November 22, 2025

Most “free” online tools monetize your data. This one doesn’t. From day one, the goal was simple: build a utility you can trust completely.

What “No Tracking” Really Means

  • No Google Analytics
  • No pixels or beacons
  • No usage logging
  • No cookies or localStorage (optional only for theme)
  • No server-side processing

How We Did It

Built with SvelteKit + adapter-static → 100% static site. Deployed on GitHub Pages. The only external request is Bootstrap CDN (for styling). Everything else — including UUID generation — happens in your browser.

You Own Your Data

Want absolute privacy? Fork the repo, disable the CDN, host it yourself. It works offline after first load.

This isn’t marketing — it’s engineering philosophy. Tools should respect users by default.
