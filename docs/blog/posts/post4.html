The Performance Stack

  • No runtime dependencies — zero npm packages at runtime
  • SvelteKit + adapter-static — compiles to pure HTML/JS
  • Tiny bundle — under 40 KB gzipped

Why It’s So Fast

No string parsing. No regex. No libraries. The core loop is just:

  • crypto.randomUUID() or getRandomValues()
  • Push to pre-allocated array
  • Render with Svelte’s ultra-efficient diffing
