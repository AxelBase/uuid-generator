Self-Hosting

Deploy Your Own Version in 60 Seconds

Published: November 22, 2025

Want absolute privacy? Don’t want to rely on anyone else’s domain? You can have your own fully functional copy running in under a minute.

One-Click Deployment

  1. Fork the repository on GitHub
  2. Go to Settings → Pages
  3. Select Deploy from a branch → main → /docs
  4. Wait 30 seconds
  5. Done — your private generator is live

Zero Maintenance

No server. No database. No updates required. GitHub Pages serves static files forever.

Go Fully Offline (Optional)

Replace the Bootstrap CDN with local files → works with no internet after first load. Perfect for air-gapped environments.

Thousands of developers already run their own instances for internal tools, compliance, or just peace of mind.
